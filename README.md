# CurrenciesCost

Представляю вашему вниманию Телеграмм-бот "SFCurriencies", который возвращает цену на определённое количество валюты (евро, доллар или рубль).
Бот в Телеграмм можно найти по имени: @CurrienciesBot
Доступны комманды: 
/start - начать работу
/values - информация о валютах, доступных для конвертации
/help - справка по работе бота
